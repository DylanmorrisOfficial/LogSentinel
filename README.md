# LogSentinel
A Python-based security log analysis platform for detecting suspicious activity and visualising security events.

## Current Status

In Development

## Planned Features

- Parse authentication and security logs
- Detect brute-force login attempts
- Identify suspicious IP activity
- Generate security alerts
- Store events in a database
- Visualise activity through a web dashboard
- Machine learning-based anomaly detection
- Docker deployment

## Technologies

- Python
- Flask
- SQLite
- Scikit-Learn
- Docker
- Pytest

## Project Goals

This project is being developed to strengthen skills in:

- Cybersecurity
- Python development
- Log analysis
- Threat detection
- Machine learning
- Software engineering practices
